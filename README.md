# .github

Community health files and the organization profile for the [Nolus Protocol](https://nolus.io) GitHub organization.

Files here act as org-wide defaults — individual repos can override by adding their own versions. See [GitHub's documentation on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for how this works.
